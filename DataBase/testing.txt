*   Python is a coding language.
*   It uses English-like words.
*   We tell the computer what to do.
*   We write instructions called code.
*   Computers follow code to do things.

*   `print("Hello, world!")`
*   This line tells Python to show text.
*   The text is "Hello, world!".
*   When run, it shows that on screen.

*   Python is a fun computer language.
*   It uses words, not just numbers.
*   You write instructions for the computer.
*   The computer then follows your instructions.
*   It's like giving commands to a robot!

*   `print("Hello, world!")`
*   This tells Python to show text.
*   "Hello, world!" is the text shown.
*   You can change the text inside "".
*   Try it out and see what happens!
